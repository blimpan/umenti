# Account Settings (`/settings`)

Accessible to both students and teachers. Contains personal profile information and account preferences.

## Layout

Persistent left sidebar. Main content area is a single scrollable column of settings sections.

## Sections

**Profile**
- Profile picture (upload or remove)
- Display name
- Email address (read-only — managed by org admin)

**About me (students only)**
- Interests — free-text field. Used by the AI tutor to personalize explanations. E.g. "Football, movies, cooking."
- A short note explaining how this information is used: "Your tutor uses this to make explanations more relevant to you."

**Security**
- Change password (current password + new password fields)

**Appearance**
- Language preference — dropdown (affects UI language, not course content language)

## What is intentionally excluded

- Notification preferences (future feature)
- Theme switching (future feature — students may eventually choose their own theme)
- Account deletion (managed by org admin)
